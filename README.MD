# 介绍

首页代码可自定义修改。

添加github贪吃蛇，侧边栏添加音乐播放器、一言、《每天60秒读懂世界》、摸鱼日历、成长轨迹……

页面主题尾部添加gif动图，可展示兴趣爱好。
